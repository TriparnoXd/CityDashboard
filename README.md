# 🌆 City Pulse

**City Pulse** is a simple and interactive app that lets you compare the **weather conditions of two cities** side by side.  
Perfect for travelers, planners, or just the weather-curious 🌍🌦️.

---

## 🚀 Features

- 🌍 Search and select **two cities** to compare.
- 🌡️ Displays **temperature, humidity, wind speed, and weather conditions**.
- 📊 Side-by-side **comparison view** for better clarity.
- 🔄 Real-time weather updates using **OpenWeatherMap API**.
- 📱 Fully responsive design (works on desktop & mobile).

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **API:** [OpenWeatherMap](https://openweathermap.org/api)  
- **Styling:** Tailwind CSS (optional, or your preferred CSS framework)
